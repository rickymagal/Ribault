# Ribault — Compiler API

HTML documentation for the compiler modules (Analysis and Synthesis). Generated with Haddock (hyperlinked source + quickjump).

Modules

  • Analysis
    Submodules
    •  Analysis.ASTGen DOT graph generator for the core AST, covering all language constructors (patterns and expressions), including TALM-style Super.
  •  Lexer
  •  Node Dataflow nodes (mirroring TALM-like mnemonics) and port helpers.
  •  Parser
  •  Port Node port abstractions (akin to the C-compiler's SteerPort / InstPort).
  •  Semantic Semantic checks (scope, arity, duplicates) and lightweight type inference over the core AST, plus desugaring and super-name assignment utilities.
  •  Syntax Core Abstract Syntax Tree (AST) for a small, strict, purely functional language with extensions for TALM-style super blocks.
  • Synthesis
    Submodules
    •  Synthesis.Builder Lowers the core AST to a dataflow graph (DFG) of DNodes, wiring ports and managing environments/aliases.
    •  Synthesis.Codegen Assemble a DGraph of DNodes into TALM/FL textual instructions.
    •  Synthesis.GraphViz Render a DGraph of DNodes to GraphViz DOT.
    •  Synthesis.SuperExtract Extracts super-instructions from the AST after assignSuperNames (names s#).
    •  Synthesis.SupersEmit Emit a Haskell module exposing s# symbols via FFI (profile B).
  •  Types Core types for the Dataflow graph generation phase.
  •  Unique Generation of fresh (unique) integers in pure and monadic code.
