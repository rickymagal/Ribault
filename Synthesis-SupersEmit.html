Synthesis.SupersEmit

Description

Generates the Haskell source for a module named Supers that exports super-instruction entry points via the FFI. Each exported symbol follows “profile B”: sN :: Ptr Int64 -> Ptr Int64 -> IO (), reading in[0] and writing out[0]. Helpers for list encoding/decoding (compatible with the Builder) are emitted alongside the supers.
