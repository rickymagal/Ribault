## Overview

This module traverses the core Syntax AST and emits a Graphviz DOT representation suitable for quick visualization and debugging. It covers **all** pattern and expression constructors currently used by the language, including the Super extension node for coarse-grained super-instructions.

## Notes

  • Node identifiers are allocated from a monotonically increasing counter kept in the state monad.
  • Edges are emitted as the traversal discovers parent–child relationships.
  • Labels are escaped to keep the DOT output valid (quotes, backslashes, newlines).

## Covered constructors

  • Patterns: PWildcard, PVar, PLit, PList, PTuple, PCons
  • Expressions: Var, Lit, Lambda, If, Cons, Case, Let, App, BinOp, UnOp, List, Tuple, Super
